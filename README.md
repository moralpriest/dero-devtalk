# DeroDevTalk

Meeting notes repository for the DeroDev Talk community.

## About

This repository contains meeting notes from DeroDev Talk community meetings, where developers and contributors collaborate on Dero-related projects.

## Meeting Files

Meeting notes are stored in the `meetings/` directory using the naming convention:

```
meetings/DeroDevTalk Meeting YYYY-MM-DD.MD
```

## Adding a New Meeting

1. Copy the meeting template or use an existing meeting file as a reference
2. Create a new file in `meetings/` with the date in the filename
3. Fill in meeting details, agenda, notes, and action items
4. Submit via Pull Request

## Structure

Each meeting file follows this structure:

- **Meeting Information** - Date, time, location, attendees
- **Agenda** - Topics to be discussed with time allocations
- **Pre-meeting Preparation** - Tasks to complete before the meeting
- **Meeting Notes** - Detailed notes from each agenda item
- **Key Decisions** - Important decisions made during the meeting
- **Action Items** - Tasks assigned to team members
